Q: What is DOM and what is the purpose of it?

Ans: DOM is a programming interface for web documents. It's full meaning is "Document Object Model". It represents the whole document as a tree and every element is a node of it. We can set class, id, attribute to the node. We can create dynamic, interactive web page by using DOM. Like we can remove or add element, styling the element of document.

Q: How to select Element by using DOM selection?

Ans: There are 5 ways to select element by using DOM selection

1. document.getElementById( )

2. document.getElementsByClassName( )

3. document.getElementsByTagName( )

4. document.querySelector( )

5. document.querySelectorAll( )

Q: What is Event bubble?

Ans: The Event bubbling in JavaScript is a type of event propagation. First the event triggers the target element the continuosly triggers the parent element of the target element. It is a procedure where it starts from the element that triggered the event and then bubbles up to the parent elements in the hierarchy.

Q: What is Callback function?

Ans: Function that is passed as an argument inside of another function is called a callback function. It can be run after another function has finished.
